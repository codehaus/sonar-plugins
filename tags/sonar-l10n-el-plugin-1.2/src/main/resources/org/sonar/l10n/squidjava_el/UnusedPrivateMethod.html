Μέθοδοι ορισμένες ως Private και οι οποίες δεν καλούνται ποτέ θεωρούνται 'νεκρός πηγαίος κώδικας'. Στην πράξη αυτό σημαίνει κώδικας που δε χρειάζεται και θα πρέπει να αφαιρεθεί. Αυτό βοηθάει στη συντήρησή του μειώνοντας τον πραγματικό κώδικα που χρειάζεται να συντηρείται,κάνοντας τον πιο κατανοητό, και εξαλείφοντας πιθανά σφάλματα πριν αυτά εμφανιστούν.

Στις δύο παρακάτω περιπτώσεις μέθοδοι ορισμένες ώς private, δεν θεωρούνται ως 'νεκρός πηγαίος κώδικας' από το Sonar :

  • Άδειοι constructors ορισμένοι ως private οι οποίοι εσκεμμένα χρησιμοποιούνται για να αποτρέψουν οποιαδήποτε άμεση αρχικοποίηση μιας κλάσης.
  • Μέθοδοι ορισμένες ως Private : readObject(...), writeObject(...), writeReplace(...), readResolve(...) οι οποίες χρησιμοποιούνται όταν η κλάση υλοποιεί το Serializable interface.
