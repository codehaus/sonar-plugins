Ο πηγαίος κώδικας συμμορφώνεται με ένα αρχιτεκτοινκό μοντέλο όταν τηρεί πλήρως ένα σύνολο αρχιτεκτονικών περιορισμών. Ένας περιορισμός επιτρέπει την απόρριψη αναφορών μεταξύ κλάσεων με τη χρήση προτύπων(patterns).

Για παράδειγμα μπορείτε να χρησιμοποιήσετε τον κανόνα ώστε να :

  • απαγορεύσετε πρόσβαση στις κλάσεις **.web.** from **.dao.**
  • απαγορεύσετε πρόσβαση στις κλάσεις java.util.Vector, java.util.Hashtable και java.util.Enumeration από όλες τις κλάσεις
  • απαγορεύσετε πρόσβαση στις κλάσεις java.sql.** from **.ui.** και **.web.**
